Tools I use & Tinker with

undergrad days

machine - ThinkPad X1 Gen 8

GNU/Linux

OpenSUSE ⧉ - I'm on the rolling release (Tumbleweed) version.

Windows is an operating system for videogames, Linux is a real operating system for serious stuff - Tsoding

Editor/IDE - Pycharm/VS Code/Sublime

I use Vim ⧉ motions to navigate and manipulate text while using any text-editor (there are plugins availabe to enable them).

I'm also considering familiarizing myself with Lua and configuring my personal Neovim ⧉ (an extended Vim fork) setup for a more customized experience.

Terminal Emulator

Kitty ⧉ - fast, feature-rich, GPU based, and highly customizable.

Shell

Fish ⧉ - a smart shell designed to have user-friendly defaults.

I like the Auto-suggestions and Tab Completion Fish offers out-of-the-box, which can save you time and help you avoid typing mistakes.

Tab Manager

Workona ⧉ - a tab manager for chromium based browsers.

Your work is scattered across dozens of tabs & cloud apps. Workona puts it all in one place, so projects are finally organized.
